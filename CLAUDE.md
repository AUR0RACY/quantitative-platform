# quantitative-platform

A quantitative trading platform — Trailblazer-inspired, built for Windows.

## Project Overview

This project aims to build a fully functional quantitative trading platform with the following core features:

- **Market Data Display**: Real-time quotes, candlestick (K-line) charts
- **Quantitative Strategy Language**: A DSL compatible with the Trader Platform ecosystem
- **Strategy Backtesting & Execution**: Write, test, and run quantitative strategies
